ESTNotification

ESTNotificationTransporter is a class that allows you to easily exchange data between host app and extension. After each successful save ESTNotification is sent so the extension or host app can read data. Before implementing this class remember to setup App Groups in your project. Please read https://developer.apple.com/library/ios/documentation/General/Conceptual/ExtensibilityPG/ExtensionScenarios.html (Sharing Data with Your Containing App section) if you have any problems with that.
